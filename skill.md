# Skill: Build an Enterprise-Level AI Consultancy Website

This skill outlines the plan to create a sophisticated, elegant, and high-performance website for an AI consultancy, drawing inspiration from top-tier sites like sanity.io.

## Project Vision

- **Goal:** To build a world-class AI consultancy website that is visually stunning, content-rich, and technically optimized.
- **Aesthetic:** Posh, elegant, and professional, with abstract videography, high-quality images, and polished "VIP-style" typography.
- **Inspiration:** sanity.io

## Phases and Tasks

### Phase 1: Foundation & Design System

1.  **Technology Stack:**
    *   **Framework:** Next.js (for performance, SEO, and server-side rendering).
    *   **Styling:** Tailwind CSS (for a utility-first approach to maintain a consistent design).
    *   **CMS:** A headless CMS (like Sanity, Contentful, or Strapi) will be used for managing content.
    *   **Deployment:** Vercel or Netlify for continuous integration and deployment.

2.  **Visual Identity:**
    *   **Color Palette:** A sophisticated and modern palette (e.g., monochromatic with a single accent color).
    *   **Typography:** Select elegant, high-quality fonts for headings and body text.
    *   **Logo:** A placeholder logo will be created.

3.  **Component Library:**
    *   Develop a set of reusable React components for consistency.
    *   **Core Components:** Buttons, Navigation Bar, Footer, Cards, Modals.
    - **Layout Components:** Main layout, page sections, grid system.

### Phase 2: Core Pages & Content Structure

1.  **Homepage:**
    *   Feature an abstract video background.
    *   A clear value proposition and call-to-action.
    *   Sections for featured services, blog posts, and client logos.

2.  **Service Pages:**
    *   Create a template for service pages.
    *   Each page will have a detailed description of the service, benefits, and case studies.

3.  **Location Pages:**
    *   Create a template for location-specific pages.
    *   Include address, map, and contact information for each office.

4.  **Blog:**
    *   Blog listing page with a clean, readable layout.
    *   Blog post page with a focus on typography and readability.
    *   Implement categories and tags for blog posts.

5.  **Entities/Case Studies:**
    *   A section to showcase successful projects or partnerships.
    *   Each entity/case study will have its own page with project details, challenges, and solutions.

### Phase 3: SEO & Performance Optimization

1.  **On-Page SEO:**
    *   **Meta Tags:** Dynamic meta titles and descriptions for all pages.
    *   **Structured Data:** Implement schema markup for services, blog posts, and locations.
    *   **Open Graph Tags:** For social media sharing.

2.  **Performance:**
    *   **Image Optimization:** Use Next.js Image component for automatic image optimization.
    *   **Video Optimization:** Ensure videos are compressed and loaded efficiently.
    *   **Code Splitting:** Lazy load components and pages.
    *   **Analytics:** Integrate web analytics (e.g., Google Analytics, Plausible).

### Phase 4: Final Touches & Deployment

1.  **Contact & Forms:**
    *   Create a contact page with a form.
    *   Integrate with an email service or CRM.

2.  **Final Review:**
    *   Cross-browser and cross-device testing.
    *   Accessibility audit.
    *   Content review and proofreading.

3.  **Deployment:**
    *   Set up production build and deployment pipeline.
    *   Configure domain and DNS settings.
    *   Monitor website performance and uptime.
